Defines subscribe/receive and publish/send remote procedure calls (in the form of messages formats) pertaining to the NuNet internal tokenomics functionality and its implementation via integration to different blockchains;

This API is intended to be full specification of functions tht enable NuNet components to negotiate, lock in smart contracts and distribute and reward each other for the computational work, data or other internal machine-to-machine services with NuNet Utility tokens (NTX).
